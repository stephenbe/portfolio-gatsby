dansmonsac.ca

Le site dansmonsac.ca est développé pour l'organisme COCQ-SIDA ayant pour but de faire de la prévention du sida au Québec. il fut de mon ressort de gérer l'intégration et l'aspect responsive du site. Ce site Web a été codé à l'aide de HTML5, CSS3 (SASS), JavaScript (jQuery). Cette réalisation est optimisée pour les téléphones mobiles et les fureteurs récents.
